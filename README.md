# 庫存系統

## Getting Started
**包含庫存及手機的 server 安裝檔，皆使用 docker 安裝 。 下面是如何安裝的步驟。**

### Prerequisites
* docker

### Installing
**1.clone repository 到 local。**
```shell
git clone https://github.com/zzhewei/bolun.git
```

**2. 建立**
```shell
docker-compose up -d
```
